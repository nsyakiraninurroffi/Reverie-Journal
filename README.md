> **Reverie** bukan sekadar platform jurnal biasa—*it's a premium, full-stack experience*. Didesain khusus untuk kamu yang mengapresiasi estetika. Dengan sentuhan antarmuka *glassmorphism* yang elegan dan animasi yang *buttery-smooth*, Reverie mengubah kebiasaan menulis jurnal menjadi sesuatu yang lebih personal, *calming*, dan *immersive*. 🌙

## ✨ What's Inside? (Features)

- **Aesthetic Glassmorphism UI:** Desain *sleek* dan modern dengan efek *frosted glass* yang *trendy* dan warna-warna yang *vibrant* tapi tetap *clean*.
- **Fluid Animations:** Didukung oleh Framer Motion untuk transisi halaman yang *spring-based* dan interaksi mikro yang bikin UI terasa *alive*. 
- **Secure Authentication:** Sistem login, registrasi, dan manajemen profil (lengkap dengan avatar) yang *robust*, di-power oleh Laravel Sanctum.
- **Rich Journaling (CRUD):** *Express yourself freely!* Buat, edit, baca, atau hapus jurnalmu. Kamu juga bisa mengunggah gambar untuk melengkapi ceritamu.
- **Social Interaction:** *Spread the love!* Berikan *like* pada jurnal *user* lain dan dapatkan notifikasi *real-time*.
- **Real-Time Feedback Loop:** Terintegrasi langsung dengan Telegram Bot API. *Admin can monitor user feedback in real-time, literally!*

## 🛠️ The Tech Stack

**Client-Side (Frontend):** React 19, Vite, Tailwind CSS v4, Framer Motion, Lucide React, Axios, React Router DOM.  
**Server-Side (Backend):** Laravel 12, PHP 8.2+, SQLite, Sanctum untuk API Authentication.  

## 🚀 How to Launch It (Getting Started)

### Prerequisites
Make sure you have these installed:
- PHP 8.2+
- Composer
- Node.js & npm

### Installation Guide

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/Reverie-Journal.git
   cd Reverie-Journal
   ```

2. **Backend Setup (Laravel vibes)**
   ```bash
   cd laravel-api-app
   composer install
   cp .env.example .env
   php artisan key:generate
   php artisan migrate
   php artisan serve
   ```

3. **Frontend Setup (React magic)**
   ```bash
   cd laravel_api_react
   npm install
   npm run dev
   ```

## 🎨 Design Philosophy & Vibe
Reverie dibangun dengan konsep **"Dark Modernity"** yang dikawinkan dengan **"Glassmorphism"**. Menggunakan tipografi *high-contrast*, *blurred backgrounds*, dan *subtle glowing gradients* untuk menciptakan ruang virtual yang terasa *personal, calm*, tapi tetap ngasih kesan *highly polished* dan profesional.
